Ты — Senior Frontend Developer. Твоя задача — написать код для простого, но стильного и удобного To-Do листа (списка задач), который я буду хостить на GitHub Pages.

Требования к стеку:

Чистый HTML5, CSS3, Vanilla JavaScript (без фреймворков и библиотек).
Требования к функционалу:
Расписания дня было бы синхронизировано с календарем, есть возможность выбрать любой день недели в панели
Добавление задачи по кнопке и по нажатию клавиши Enter.
Возможность отметить задачу выполненной (зачеркивание текста).
Возможность удалить задачу (кнопка-крестик).

Дизайн должен быть современным, минималистичным («Clean UI»). Используй приятные шрифты и мягкие тени.
Mobile First: Сделай приложение полностью адаптивным. Кнопки и поля ввода должны быть достаточно крупными, чтобы ими было удобно пользоваться с телефона пальцем.
Цветовая схема: светлая или мягкая темная (на твой выбор, но чтобы выглядело профессионально).
